# Student Average Calculator 

## Description
This project is a Python program that calculates a student´s average grade.
It is a simple academic tool designed to collect and organize grades from different categories.

The program is structured to store grades using a dictionary
It is useful for demonstrating basic Python concepts such as user input, dictionaries, loops, conditionals, and error handling.

## HOW IT WORKS?
1. The program asks the user to enter the student´s name.
2. The program requests three grades: activity, homework, and exam.
3. It stores the grades inside a dictionary called as the same thing 'grades'.
4. The program calculates the average grade using the `sum()` function and the total number of grades.
5. A ´for´ loop prints each categories and its corresponding grade.
6. The program checks the average grade and dtermines if the student passes or fails.
7. If the user enters an invalid value, the program shows an error message using a `try` and `except` block.
